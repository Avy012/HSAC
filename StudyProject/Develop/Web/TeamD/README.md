## **🚩 Plan & Rule**

    🔊 스터디 방식
        1. 선 학습 후, 모든 팀원이 발표 준비를 한다.
        2. 스터디 당일, 랜덤으로 발표자를 선발한다.
        3. 발표자는 해당 학습 분량에 대하여 발표를 진행한다.
        4. 다른 팀원들은 해당 분량에 대하여 질문/피드백을 진행한다.
        5. 스터디 종료 후 다음 진도를 선정한다.
