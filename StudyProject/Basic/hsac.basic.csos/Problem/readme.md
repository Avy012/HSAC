<p><b>문제 :</b></p>
<p>자연수 N이 주어진다. N을 이진수로 바꿔서 출력하는 프로그램을 작성하시오 </p>
<p><b>출제 이유 :</b></p>
<p>10진 정수를 2진수로 바꾸는 것은 상당히 중요한 알고리즘이라고 생각해 스터디원과 공유하고자 해당 문제를 출제하게 되었습니다.</p>
<p><b>답안 : </b></p>

```
void ten_to_two_j(unsigned n) {
    if (n < 2) { // n==1 (이전 n이 2이거나 3)
        printf("%d", n);
    }
    else {
        ten_to_two_j(n / 2);
        printf("%d", n % 2);
    }
}
```
